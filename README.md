Portfolio Website

A personal portfolio website built with HTML, CSS, JavaScript, Python, and Django to showcase my projects, skills, and experience as a web developer. The website features a modern, responsive design with smooth interactions and a professional layout.

Features:

Clean, modern, fully responsive design
Project showcase section with live links to deployed projects
About section highlighting skills and background
Contact section with form and links for easy communication
Smooth scrolling and interactive elements powered by JavaScript

Purpose:

This website was created to professionally present my work, demonstrate frontend skills, and serve as a central hub for all my projects.
